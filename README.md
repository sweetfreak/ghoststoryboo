# A Terrifying Ghost Story!

An interactive ghost story built entirely in JavaScript and CSS.

## Built With
- JavaScript
- CSS

## Get started
Play this game at [Ghost Story Boo](https://sweetfreak.github.io/ghoststoryboo/)

## Features
A simple choose your adventure story that uses JavaScript to adjust slides. All animations and transitions were made with CSS. Users can continue where they left off, thanks to LocalStore features.
This game also features artwork by [Arthur Hickman](https://www.instagram.com/arthurmuscles/?hl=en)
